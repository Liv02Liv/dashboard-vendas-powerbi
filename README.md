# 📊 Dashboard Comercial – Indicadores de Vendas

Projeto desenvolvido para análise de desempenho comercial de uma loja de calçados sustentáveis.

## Objetivo
Analisar indicadores de vendas e desempenho de produtos e vendedores.

## Indicadores analisados
- Faturamento total
- Evolução das vendas por dia
- Vendas por produto
- Contribuição por vendedor

## Ferramentas utilizadas
- Power BI
- Excel
- Modelagem de dados
